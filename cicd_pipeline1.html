JENKINS
Automation Server
  • Webhook triggers on git push event
  • Declarative Jenkinsfile pipeline-as-code
  • Parallel test execution across agents
  • SonarQube quality gate enforcement
  • Shared library for reusable steps
  • Slack/email alert on failure
01
DOCKER
Container Build
  • Multi-stage Dockerfile minimizes size
  • Secrets injected at build-time only
  • Trivy CVE scan before push
02
REGISTRY
Image Management
  • SHA + build-number tag strategy
  • Image signing via cosign
  • Retention policy for old images
03
KUBERNETES
Cluster Deployment — Rolling Update
  • kubectl apply from updated manifest
  • maxUnavailable: 0 — zero downtime
  • Readiness probe gates traffic shift
  • HPA auto-scales on CPU above 70%
  • Rollback on failed health check
  • Namespace isolation per environment
